# Chrys-Smart-Expense-Tracker.cli
A smart and very useful expense tracker cli that stores all the expenses of the client.

What It Does
-Keeps track of all expenses
-Allow deleting,editing and viewing of expenses
-Shows your total against your set budget
-Feedback option
-About section
-Allows quitting at anytime

How To Run.
-Make sure python is installed and run
Smart Expense Tracker.cli
